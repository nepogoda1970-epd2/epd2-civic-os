# Contracts

Заготовка для будущих версионированных контрактов между модулями EPD²
Civic OS. На этапе CLAUDE-PACK-01 контракты не содержат ни одной реализации
— только структуру каталогов и документированный список reason codes.

## Каталоги

- `openapi/` — будущие версионированные OpenAPI-спецификации сервисов.
- `events/` — будущие версионированные схемы канонических событий (раздел
  20–21 `docs/canonical/TZ-00-domain-event-canon.md`).
- `schemas/` — будущие JSON Schema / Pydantic-описания канонических
  сущностей.
- `reason-codes/` — документированный список reason codes из ТЗ-00 (раздел
  24 канона). Пока не исполняемый enum.
- `fixtures/` — будущие тестовые фикстуры для contract tests (раздел 27
  канона).

## Правило версионирования

Любое изменение контракта в этих каталогах версионируется. Обратно
несовместимое изменение требует новой major-версии, а не молчаливой замены
смысла существующей версии (см. `CONTRIBUTING.md`).
