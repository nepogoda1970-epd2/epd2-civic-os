# web-shell

Минимальный Next.js frontend-каркас EPD² Civic OS.

## Назначение

Показать единственную статическую страницу со статусом инфраструктурного
каркаса. Ничего больше.

## Ограничения этого пакета

- без API-маршрутов;
- без login/аутентификации;
- без форм;
- без политического контента;
- без сторонних UI-библиотек (никакого component kit);
- без сложного дизайна (только минимальные, базовые стили);
- без аналитики;
- без cookies;
- без внешних шрифтов (используется системный font stack).

## Запуск

```bash
npm install
npm run dev      # локальная разработка
npm run build    # production build
npm run test     # smoke test
npm run lint     # ESLint
npm run typecheck
```

## Тесты

`tests/smoke.test.ts` проверяет, что главная страница содержит заголовок
`EPD² Civic OS`.
