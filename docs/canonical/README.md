# docs/canonical

Каноническая документация EPD² Civic OS.

## Содержимое

- `TZ-00-domain-event-canon.md` — ТЗ-00, каноническая модель домена и
  событий. Действующий архитектурный канон проекта, версия `0.2.0`.
- `canon-version.json` — машинно-читаемая версия канона и диапазон
  совместимости репозитория.

## Правило неизменности

Файл `TZ-00-domain-event-canon.md` **нельзя**:

- редактировать;
- сокращать;
- переименовывать сущности, статусы, события или reason codes в нём;
- изменять описанные архитектурные инварианты,

без отдельного задания и принятого `Architecture Decision Record` (см.
`docs/adr/`).

Любое предложение по изменению канона фиксируется в
`docs/review/OPEN_QUESTIONS.md` или оформляется как issue по шаблону
`architecture_change.yml`, но не вносится в канон самостоятельно.
